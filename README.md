🎬 Movie Show Application
Welcome to the Movie Show Application! This is a React.js-based web application where users can explore trending movies, browse through pages for more movies, and create a personalized Watchlist.

🌟 Features
Home Page
Displays trending movies fetched from an external API.
Users can navigate through multiple pages to explore more movies.
Watchlist
Add your favorite movies from the Home Page to the Watchlist.
The Watchlist allows users to:
Filter movies by genre.
Sort movies by rating.
Search for specific movies.
Watchlist persists even after refreshing the page, thanks to Local Storage.
React Hooks
useState: To manage the state of movies, watchlist, and filters.
useEffect: To fetch data from the API and handle updates dynamically.
🚀 Technologies Used
React.js
HTML
CSS
JavaScript
Tailwind CSS
🛠️ Functionalities
API Integration: Used an API to fetch trending movie data and display it on the home page.
Watchlist Management: Users can manage their Watchlist, which includes features like filtering, sorting, and searching.
State Management: Used React hooks (useState, useEffect) for dynamic state handling.
Persistent Data: Leveraged Local Storage to ensure movies remain in the Watchlist even after refreshing the site.
